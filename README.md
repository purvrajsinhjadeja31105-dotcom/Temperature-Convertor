# Temperature-Convertor
A simple Temperature Converter built using HTML, CSS, and JavaScript. This application allows users to convert temperature values between different units such as Celsius, Fahrenheit, and Kelvin.

## Features:

Convert between Celsius, Fahrenheit, and Kelvin
Real-time calculation on user input
Simple and intuitive user interface
Responsive design

## Tech Stack:

HTML
CSS
JavaScript

Purpose:
Developed to strengthen understanding of JavaScript logic, DOM manipulation, and handling user input in web applications.
